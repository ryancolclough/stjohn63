# CORE Platform 1.2.2-dev

Build: `20260712.001`  
Release ID: `CORE-DEV-REL-003`

## Added
- Full Amendment Centre
- Independent amendment workflow stages
- Amendment IDs
- Current wording and proposed wording comparison
- Committee recommendation
- Motion and meeting records
- Moved-by and seconded-by fields
- Recorded vote counts and vote result
- Approval date
- Publication readiness validation
- Ready-to-publish dashboard count
- Approved-only public amendment export
- Event bus return values for module-to-module data collection

## Module Versions
- Dashboard `1.2.0`
- Review Engine `1.3.1`
- Amendment Engine `1.2.0`
- Export Engine `1.3.0`
- Settings `1.2.0`

## Important
CORE does not approve an amendment. It records and validates the Temple Board's approval process.
