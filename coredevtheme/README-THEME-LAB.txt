CORE THEME LAB 1.7.0

Folder name:
coredevtheme

This is a separate visual branch based on the stable CORE 1.6.2 release.

Included profiles:
1. Alpine Night — default
2. Titanium
3. Heritage

Appearance controls:
- Light / Dark / System
- Crystal / Balanced / Frosted glass
- Off / Low / Normal ambient motion

Nothing in this package changes your stable CORE folder or its saved data.
Because the folder name is different, browser storage for this branch may also
be separate from the production CORE folder depending on the browser.
